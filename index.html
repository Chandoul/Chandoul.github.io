Age of Empires II: AoK & AoC Easy Manager

Humbly made by Smile 💚





How to use it

Assuming you downloaded and installed the application correctly on your computer.

My Game

  • Select your game location locally.
  • Open the selected location.
  • Checks the game location in your computer and tries to load it if there is any.
  • Asks for a download location then starts downloading the game.
  • Selects which game, Confirm uninstall.
  • Finally an optional choice is to choose to update your game desktop icons, keep it on in order to update them if you use multiple game locations.

Version

  • Apply a version for Age of Empires II The Age of Kings.
  • Apply a version for Age of Empires II The Conquerors.
  • Apply a version for Age of Empires II The Forgotten Empires.

Patch/Fix

  • Clear a fix if there is any.
  • Apply a fix for Age of Empires II.
  • In the right side, you will find some options regarding those fixs, those fix generally do the following enhancement:
    • Widescreen (Forces your desktop resolution in the game)
    • Overlay tech tree (Shows a resume in icons equivalent to your civlization)
    • Villager per resources (Wood, Food, Gold, Stone, villager count displayed next to the resources icons)
    • Water animation (Seas and lakes are more realistic)
    • Colored signal (Signals shown in colors equivalent to each player)
    • Auto farm queue (Villager automatically reseed their farms)
    • MQ (Multiple Queue, helps in creating units from multiple selected building with one click)
    • Civilization easy display (Displays civilization name next to each score)
  • Includes all those features, so it is highly recommended to use it, because the lower version fixs may miss some those enhancement

Language

  • Changes the game interface language.

Visual mods

  • These mods are installed locally, means other players do not need to have them if you did install any of them.
  • Installs a visual mod.
  • Uninstalls a visual mod.
  • Of course this is not perfect because I didn't include all visual mods out there, instead only the most important ones, plus if you wish to have your owns mods just let me know, and then I will see if I can add them.

Data mods

  • Unlike the visual mods, these mods must be installed globally, means other players must have them if you did install any of them.
  • Installs a data mod.
  • Uninstalls a data mod.
  • Of course this is not perfect because I didn't include all data mods out there, plus if you wish to have your owns mods just let me know, and then I will see if I can add them.

Hide All IP trial resetter

  • This is a vulnerablity, clears the registery so you can have new trial period of two days.
  • There are 6 attemps to achive this, if none of them worked, try the same process again.

Shortcuts / Keys remapper

  • If you got some knowledge with AutoHotkey scripting language you can bind or remap your keyboard keys to fire some other custom keys.
  • Currently there are two custom options:
    • + Exits the game, in case you got stucked.
    • + Unselects one unit on a group selection and sends a right click.

Direct Draw Fix

  • An engine created in order to run old games in the correct setting in the recent copy of windows.
  • This fix is very important if you want to have the best game experience
  • Applies direct draw fix.
  • Removes direct draw fix.
  • Opens the Age of Empires II Direct Draw options window.
  • Opens The Conquerors Direct Draw options window.

GameRanger Account Switcher

  • Adds an already logged account to the account list.
  • Removes an already added account to the account list.
  • Login with the selected account from the list.
  • Restarts GameRanger.
  • Launchs the process where you can create a new GameRanger account.

Scenarios

  • Based on a friend request I have added this option to extract Hints and many other info from scenario files for a quick access of the correct game setting.
