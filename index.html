How to use it

Assuming you downloaded and installed the application correctly on your computer.

[My Game] section

  • Select your game location locally by clicking on the button.
  • You can optionally open the selected location using button.
  • If you are not sure what to select you can click on button, which will check the game location in your computer and try to load it if there is any.
  • Now if you don't have the game and you need to download it, no problem, all you need to do is click on button, which will ask for a download location then it will start working on it.
  • To remove any game you have downloaded click on button, select which game location then click .
  • Finally an optional choice is to choose to update your game desktop icons, keep it on in order to update them if you use multiple game locations.
